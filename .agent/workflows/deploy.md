---
description: How to deploy the Mongo API to Render and MongoDB Atlas
---

1. Ensure your code is pushed to a GitHub repository.
2. Follow the steps in the [deployment_guide.md](file:///C:/Users/Maurya/.gemini/antigravity/brain/40eb5748-533d-478f-b91f-78f77d184a24/deployment_guide.md) to set up MongoDB Atlas.
3. Link your GitHub repo to [Render](https://render.com/).
4. Configure the environment variables `MONGO_URI` and `PORT` on Render.
5. Deploy and monitor the logs on Render dashboard.
